Description

The purpose of the app is to graphically show how the distribution of sample mean becomes approximately normal as more random samples are drawn from the original population. You can experiment with a variety of population distribution shapes and sample sizes.

Parameters

Shape

a list of available population distribution shapes to select from.

N

number of observations in the population.

Mean, Std. Deviation, Min, Max, Degrees of Freedom, Rate

arguments necessary to define the population distribution. Each population type is defined by an specific set of arguments.

Size

number of observations drawn on each sample.

Repeats

number of Size repetitions to draw at the same time.

Set Random Seed

enable the use a custom random seed to have repeatable results between sessions.

Random Seed

number to set as random seed at the beginning of the simulation.

Variable Name

name of the population variable.

Summary Decimals

maximum number of decimals to show at summary statistics.

Buttons

Draw Samples

click to draw samples from the population of size Size by Repeats times.

Reset

click to restart the simulation.

Help

click to show help window.
